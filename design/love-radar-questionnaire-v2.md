# 🫶 「恋爱雷达」— 恋爱风格测评 v2（情景版）

> 改版思路：全部采用情景式题目，每题是一个会心一笑的恋爱场景，A/B/C 三个选项对应不同恋爱反应。

---

## 维度体系（不变）

5 组 × 3 维度 = 15 维，评分范围 -2 ~ +2。

| 组 | 代码 | 维度名 |
|----|------|--------|
| 速度 Tempo | T1 | 节奏偏好（按部就班 ↔ 随心所欲） |
| | T2 | 投入方式（渐进 ↔ 快速上头） |
| | T3 | 确认需求（需要名分 ↔ 不需要定义） |
| 距离 Distance | D1 | 独立需求（需要空间 ↔ 喜欢黏着） |
| | D2 | 情感交融（保持自我 ↔ 完全融入） |
| | D3 | 社交边界（各自隐私 ↔ 全面透明） |
| 温度 Temp | E1 | 表达方式（含蓄 ↔ 直接） |
| | E2 | 安全感来源（来自自己 ↔ 来自对方） |
| | E3 | 冲突处理（回避 ↔ 正面解决） |
| 模式 Mode | M1 | 关系态度（轻松随性 ↔ 严肃认真） |
| | M2 | 专一度（开放探索 ↔ 绝对专一） |
| | M3 | 未来导向（活在当下 ↔ 长远规划） |
| 人设 Role | R1 | 关系角色（被照顾 ↔ 照顾对方） |
| | R2 | 情绪主导（理性 ↔ 感性） |
| | R3 | 自我风格（保持真实 ↔ 为关系调整） |

## 20 道情景题

### Q1 — 节奏偏好 / 投入方式

```
🔥 场景：你在某个app上和一个陌生人聊得飞起，
连续三天从早安聊到晚安。
第四天对方说："周末有空见一面吗？"
A) "有！时间地点你定"                                     → T1+1, T2+1
B) "聊得挺好的，但再聊一阵吧，不着急"                    → T1-1, T2-1
C) "周末暂时不确定，下周看看"（真实原因：还没做好心理准备）→ T1+1（偏消极）, T2 无变化
```

### Q2 — 确认需求 / 专一度

```
🔥 场景：你跟一个人暧昧了两个月，该聊的都聊了，
该约的也约了，但对方始终不说清楚你们是什么关系。
今天你刷到 TA 发了一条朋友圈，配文"单身真好"。
A) 截图发给闺蜜："他是不是有什么大病？"                   → T3+2, M1+1
B) 无所谓，反正我也在跟别人聊                            → T3-2, M2-2
C) 心里一沉，但不会问，默默减少主动发消息的频率           → T3-1, E2-1
```

### Q3 — 独立需求 / 情感交融

```
🔥 场景：你本来计划好周六一个人在家瘫一天——
点上外卖、开一部电影、谁都别烦我。
对象周五晚上说："明天天气超好，我们去xx玩吧！"
A) "好啊！跟你一起更开心"                                → D1-1, D2+1
B) "今天想自己待着，明天再找你好不好"                    → D1+2
C) 答应了，心里默念"我的独处日又泡汤了"                  → D1+1, R3+1
```

### Q4 — 社交边界

```
🔥 场景：你跟对象在一起半年了，感情不错。
有一天 TA 说："我们换手机用一天吧，看看对方手机里有什么。"
A) "好啊！你先看我的"                                    → D3+2
B) "不行，手机是个人隐私，跟爱不爱没关系"                → D3-2
C) "可以啊，但你先说你想找什么"                          → D3-1, E2+1
```

### Q5 — 两个约会对象撞车（用户提供的思路）

```
🔥 场景：你最近在接触两个人，A 和 B。
因为一个不可抗力的乌龙（你记错/约重复/两人都有票），
A 和 B 同时出现在了同一个地方，而且互相看到了对方和你。
A) 大方打招呼："好巧，介绍一下，这是我朋友A，这是我朋友B" → M2-1, R3+2
B) 手心冒汗，假装接电话先撤一步                          → E3-1, R1-1
C) 心态稳住，该聊聊该笑笑，等后续再分别解释              → E1+1, R3 无变化
```

### Q6 — 表达方式 / 温度

```
🔥 场景：你很想一个人。打开对话框，上一条消息还是 TA 发的，
你回复了但 TA 没再回。现在是晚上 11 点。
A) 直接发："我想你了"                                    → E1+2
B) 翻了翻 TA 的朋友圈，默默关掉手机睡觉                  → E1-2
C) 分享一首歌到朋友圈，希望 TA 能懂                      → E1-1（含蓄式表达）
```

### Q7 — 安全感来源

```
🔥 场景：暧昧对象今天一整天没找你。
你发了一条消息过去，过了 3 小时，已读不回。
A) 开始复盘自己昨天是不是说错了什么话                    → E2+1, R2+1
B) 无所谓，TA 肯定在忙，忙完了会找我的                  → E2-2
C) 去翻 TA 的社交动态，试图找到 TA 在干嘛的线索          → E2+1
```

### Q8 — 冲突处理

```
🔥 场景：你和对象因为一件小事吵起来了。
对方越说越激动，声音越来越大。
A) "我们先冷静一下再聊"——转身去阳台                     → E3-1
B) 大声回去："你能不能不要每次都这样！"                   → E3+2, R2+1
C) 不说话，等 TA 说完了再说                             → E3-2
```

### Q9 — 关系态度

```
🔥 场景：你和一个各方面条件都很合适的人约会了三个月，
一切都很好——但你总觉得少了点心动的感觉。
对方认真地问你："你觉得我们有未来吗？"
A) "对我来说心跳比条件重要，对不起"                      → M1-1, R2+1
B) "合适比心动更重要，感情可以培养"                      → M1+2, M3+1
C) "我还不太确定，能再给点时间吗"                        → 无变化
```

### Q10 — 未来导向

```
🔥 场景：你在朋友聚会上认识了一个很聊得来的人。
临走前对方说："这周末有个展，要不要一起？"
A) "好！周六还是周日？我都可以"——当场定下来              → M3+1, T1+1
B) "看情况吧，到时候再说"——结果真的到时候也没再联系      → M3-1, T1-1
C) "加个微信，你定好时间告诉我"                          → 无变化
```

### Q11 — 专一度（重口味情景）

```
🔥 场景：你发现你的暧昧对象在跟你聊的同时，
还用同一个梗（连表情包都一样）在跟你的朋友聊。
A) 直接问 TA："你是不是也在跟xx聊？"                     → M2+2, T3+1
B) 不愧是 TA，同时控场能力很强——但你要不也再加两个人？    → M2-2, R3+1
C) 默默下头，但不质问，慢慢冷掉这段关系                   → E3-1, M2+1
```

### Q12 — 关系角色

```
🔥 场景：你们约好了一顿重要的晚餐。
对方说："你定吧，我都可以。"
A) 你直接选好餐厅、订好位、规划好路线                    → R1+2
B) "我选怕你不喜欢，还是你定吧"                          → R1-2
C) "那我说几个选项，你挑一个"                            → 无变化
```

### Q13 — 情绪主导

```
🔥 场景：你朋友的恋爱故事讲给你听——
TA 对象绿了 TA，但 TA 还在纠结要不要原谅。
而你越听越气，比 TA 还激动。
A) 正常，因为我谈恋爱也是走心不走脑的                    → R2+2
B) 会帮 TA 冷静分析：绿了一次大概率有第二次              → R2-2
C) 先陪 TA 骂，骂完了再理性聊                            → 无变化
```

### Q14 — 自我风格

```
🔥 场景：对象约你去一个很高档的餐厅吃饭。
你衣柜里的风格是卫衣牛仔裤。你会：
A) 平时穿什么就穿什么，TA 认识我的时候我就是这个样子     → R3-2
B) 特意买/借一身配得上的行头                              → R3+2
C) 稍微打扮一下，但不会太夸张                            → 无变化
```

### Q15 — 前任出现

```
🔥 场景：你跟现任在商场逛街，迎面走来了你的前任。
前任看了你一眼，又看了现任一眼。
A) 大方打招呼："嗨，好巧。这是我对象xx"                  → E1+1, M2+1
B) 假装没看见，低头玩手机绕过去                          → E3-1
C) 简单点个头，然后拉着现任走开                          → 无变化
```

### Q16 — 节奏/投入

```
🔥 场景：你跟一个人约会了三次，感觉都挺好。
第三次约会结束，对方在送你回家的路上说：
"我觉得我们挺合适的，要不交往试试？"
A) "我也觉得！好！"                                      → T2+1, M1+1
B) "让我再想想"——其实是想再观察一阵                      → T2-1, T1-1
C) "先处着看看吧"——不拒绝也不正式答应                    → T3-1, M1-1
```

### Q17 — 情感交融 / 社交边界

```
🔥 场景：你最好的朋友偷偷告诉你，
TA 觉得你的对象有问题，劝你慎重。
A) 相信朋友的眼光，回去跟对象好好聊一下                  → D2-1, E3+1
B) 有点不高兴："你不了解 TA，别乱说"                     → D2+1, M2+1
C) 把朋友的话放在心里，开始更仔细观察对象的行为           → E2+1
```

### Q18 — 隐藏触发题 ①

> 触发条件：Q2 选 B、Q11 选 B

```
🔥 （隐藏）场景：你朋友突然问你——
"你现在同时对几个人有好感？"
A) "说实话，不止一个"                                    → 隐藏人格检测继续
B) "没有吧，就是随便聊聊"                                → 隐藏人格检测继续
C) "就一个，我很专一的"                                  → 不触发隐藏人格分支
```

### Q19 — 隐藏触发题 ②

> 触发条件：Q18 选 A/B

```
🔥 （隐藏）场景：深夜两点，你刷着手机突然觉得——
"我好像也没那么想谈恋爱。"
这种感觉多久出现一次？
A) 经常，间歇性想寡着                                   → 隐藏人格概率 +60%
B) 偶尔，但见到心动的人又好了                            → 隐藏人格概率 -20%
C) 几乎不，我一直想谈                                    → 不触发隐藏
```

### Q20 — 隐藏触发题 ③

> 触发条件：Q19 选 A

```
🔥 （隐藏）场景：说真的，你是怎么看到这个测试的？
A) 朋友分享给我的，说"超准你快测"                        → 隐藏人格概率 +80%
B) 自己刷到的，好奇点进来                                → 隐藏人格概率 +30%
C) 就……无聊呗                                            → 隐藏人格概率 +50%
```

**一句话人设：** 你是相亲市场的优等生——学历工作家庭爱好全打勾，七大姑八大姨提起你都竖大拇指。你相信恋爱是讲条件的，合适比心动重要。但你偶尔也会想：如果把那些条条框框拿掉，我还会不会爱？

**与相近类型的区别：**
- vs 🕰️ 老派约会选手 → 老派是浪漫复古，适配者是务实打分
- vs 📋 程序猿恋爱 → 程序猿是习惯性流程化，适配者是社会期待内化
- vs 🧊 理性派 → 理性派为自己算账，适配者为别人眼中的标准打卡
