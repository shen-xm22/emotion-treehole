# 🫶 「恋爱雷达」— 恋爱风格测评 问卷设计 v1

> 设计目标：趣味娱乐向，非心理学量表，适合截图分享和社交传播。
> 对标 SBTI 模式：低门槛 × 自嘲标签 × 隐藏彩蛋 × 高分享欲

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 测试名称 | 恋爱雷达 — 你的恋爱风格鉴定 |
| 题目数 | 20 题（17 道常规 + 3 道隐藏触发题） |
| 答题方式 | A/B/C 三选一 |
| 预计时长 | 3-5 分钟 |
| 维度体系 | 5 组 × 3 维度 = 15 维 |
| 人格数量 | 16 种（15 种常规 + 1 种隐藏） |
| 结果形式 | 人格名 + 一句梗描述 + 15 维雷达图 + 你的恋爱画像小作文 |

## 二、维度体系

每个维度评分范围 **-2 ~ +2**（-2=极低，+2=极高），共同构成用户的 15 维向量。

### 组 1：速度（Tempo）— 恋爱节奏

| 维度 | 代码 | -2 倾向 | +2 倾向 |
|------|------|---------|---------|
| 节奏偏好 | T1 | 随心所欲，按感觉走 | 按部就班，有节奏感 |
| 投入方式 | T2 | 渐进投入，慢慢来 | 快速上头，一秒沦陷 |
| 确认需求 | T3 | 不需要名分，开心就行 | 必须定义关系才安心 |

### 组 2：距离（Distance）— 亲密空间

| 维度 | 代码 | -2 倾向 | +2 倾向 |
|------|------|---------|---------|
| 独立需求 | D1 | 高度独立，需要大量独处 | 粘人，恨不得一直贴贴 |
| 情感交融 | D2 | 保持自我边界清晰 | 喜欢完全融入对方 |
| 社交边界 | D3 | 各自留私人空间 | 生活全面公开透明 |

### 组 3：温度（Temperature）— 表达与安全感

| 维度 | 代码 | -2 倾向 | +2 倾向 |
|------|------|---------|---------|
| 表达方式 | E1 | 含蓄克制，浓情于心 | 直接热烈，爱要大声说 |
| 安全感来源 | E2 | 安全感来自自己 | 安全感来自对方确认 |
| 冲突处理 | E3 | 回避冷处理，需要空间 | 正面解决，必须说清楚 |

### 组 4：模式（Mode）— 关系态度

| 维度 | 代码 | -2 倾向 | +2 倾向 |
|------|------|---------|---------|
| 关系态度 | M1 | 轻松随性，不搞沉重 | 严肃认真，以终为始 |
| 专一度 | M2 | 开放探索，不设限制 | 绝对专一，身心同步 |
| 未来导向 | M3 | 活在当下，不谈以后 | 长远规划，想做长期 |

### 组 5：人设（Role）— 角色定位

| 维度 | 代码 | -2 倾向 | +2 倾向 |
|------|------|---------|---------|
| 关系角色 | R1 | 喜欢被照顾，被动方 | 喜欢照顾人，主动方 |
| 情绪主导 | R2 | 理性主导，用脑恋爱 | 感性主导，用心恋爱 |
| 自我风格 | R3 | 在关系中保持真实自我 | 会为关系调整扮演角色 |

## 四、题目设计（20 题）

> 每道题 **A/B/C 三选一**。标注了每道题影响的维度及得分变化。

### 第 1 题：节奏

```
💬 你刚认识一个很聊得来的人，对方约你周末出去，你会：
A) 行啊，正好有空，去吧         → T1+1, T2+1
B) 先聊一阵再说，不着急见       → T1-1, T2-1
C) 看他约什么，合适就去          → 无变化（中庸选项）
```

### 第 2 题：投入方式

```
💬 心动对你来说通常是什么样的？
A) 一秒上头，第一眼就知道有没有  → T2+2
B) 慢慢相处，日久生情            → T2-2
C) 看情况，有的人一眼有的人要时间 → T2 无变化
```

### 第 3 题：确认需求

```
💬 暧昧了两个月，对方一直不挑明关系，你会：
A) 忍不住先开口，要个准信        → T3+2, M1+1
B) 挺好的，就这样处着别定义      → T3-2, M1-1
C) 心里有点难受但不会主动说      → T3 无变化, E2-1
```

### 第 4 题：独立需求

```
💬 周末你计划好一个人在家打游戏/追剧，对象突然说想来找你：
A) 来啊！一起玩更开心            → D1-1, D2+1
B) 委婉拒绝，说今天想一个人待着   → D1+2
C) 会答应但心里有点遗憾独处时间没了 → D1+1
```

### 第 5 题：情感交融

```
💬 你觉得最健康的恋爱状态是：
A) 两个人像两棵树，并肩生长但各自扎根 → D2-2, D3-1
B) 两个人像两团泥，捏在一起分不清你我 → D2+2, D3+1
C) 有时候黏在一起，有时候各自独立     → 无变化
```

### 第 6 题：社交边界

```
💬 你会告诉对象你的手机密码吗？
A) 主动告诉 TA，没什么不能看的     → D3+2
B) 不给，各自保留隐私空间          → D3-2
C) TA 要就给，但不会主动给        → D3 无变化
```

### 第 7 题：表达方式

```
💬 你想对方了，你通常会：
A) 直接发消息说 "我想你了"       → E1+2
B) 憋着，等对方先来找我          → E1-2
C) 发个表情包或者分享个视频暗示   → E1 无变化
```

### 第 8 题：安全感来源

```
💬 对象超过 4 小时没回消息，你第一反应是：
A) 有点慌，在想是不是自己做错了什么 → E2+1, E1+1
B) 无所谓，肯定在忙，忙完会找我的   → E2-2
C) 去翻一下 TA 的朋友圈/社交动态    → E2+1, E1-1
```

### 第 9 题：冲突处理

```
💬 吵架了，你通常的第一反应是：
A) 想马上说清楚，不解决问题睡不着  → E3+2, M1+1
B) 需要冷静一下，过几个小时再说    → E3-1
C) 冷战，等对方先低头             → E3-2, R1+1
```

### 第 10 题：关系态度

```
💬 你觉得谈恋爱应该：
A) 奔着结婚去的，不然浪费时间     → M1+2, M3+2
B) 享受当下就好，以后的事以后再说  → M1-2, M3-2
C) 看人，对的人自然就走到以后了    → 无变化
```

### 第 11 题：专一度

```
💬 你觉得同时跟 2-3 个人保持暧昧正常吗？
A) 很正常，没确定关系前都是自由的  → M2-2
B) 不太能接受，认定一个人就会专注  → M2+2
C) 能理解别人这样，但自己做不太到  → M2 无变化
```

### 第 12 题：未来导向

```
💬 约会时你会主动规划下次见面的时间地点吗？
A) 会，走之前就约好下次           → T1+1, M3+1
B) 不会，随缘，到了再说           → T1-1, M3-1
C) 会问对方有没有想法，但自己不主动定 → 无变化
```

### 第 13 题：角色定位

```
💬 在关系中你通常更倾向于：
A) 照顾对方，安排一切             → R1+2
B) 被照顾，喜欢对方拿主意         → R1-2
C) 互相照顾，看情况               → 无变化
```

### 第 14 题：情绪主导

```
💬 做恋爱相关决定时，你更多是：
A) 跟着感觉走，心动了就是理由     → R2+2, M1-1
B) 理性分析，合不合适心里有数     → R2-2, M1+1
C) 两者都有，看具体什么事         → 无变化
```

### 第 15 题：自我风格

```
💬 跟对象出门约会，你会穿平时自己的风格还是特别打扮？
A) 平时穿什么就穿什么，舒服最重要  → R3-2
B) 会特意打扮，给 TA 留个好印象    → R3+2
C) 稍微收拾一下但不会大变样       → 无变化
```

### 第 16 题

```
💬 你朋友的恋情出了严重危机，TA 来找你倾诉，你会：
A) 认真听，给 TA 出主意          → R1+1, E1+1（偏向治愈系）
B) 陪 TA 吐槽发泄，但不说该不该分 → 无变化（中庸选项）
C) 建议 TA 冷静分析一下利弊       → R2-1, M1+1（偏向理性派）
```

### 第 17 题

```
💬 你觉得最能代表你恋爱方式的一句歌词/台词是：
A) "慢慢来，比较快"              → T1+T2+T3 趋中，偏慢
B) "爱是折磨人的东西"            → E1+1, R2+1, E3 趋极
C) "有些人说不出哪里好，但谁都替代不了" → M2+M3 偏高
```

> （此题偏向投射型，各选项对多重维度有轻微影响）

### 第 18 题（隐藏触发①）

> 仅当 第 3 题选 B、第 11 题选 A、第 12 题选 B 时出现

```
💬（隐藏）🤫 老实说，你现在是不是：
A) 单身且享受单身                 → 触发隐藏人格检测
B) 有在接触的人但没多想           → 触发隐藏人格检测
C) 有对象了但就是好奇测一测       → 不触发隐藏人格
```

### 第 19 题（隐藏触发②）

> 仅在第 18 题选 A/B 后出现

```
💬（隐藏）你上次心动是什么时候？
A) 太久远了，记不清了              → 隐藏人格概率 +50%
B) 最近确实有个人让我有点在意      → 隐藏人格概率 -20%
C) 每天都在心动（墙头多            → 不触发隐藏人格
```

### 第 20 题（隐藏触发③）

> 仅在第 19 题选 A 后出现

```
💬（隐藏）说真的，你测这个是不是因为：
A) 朋友发了朋友圈让我也来测测     → 隐藏人格概率 +80%
B) 就是想看看自己是什么类型        → 隐藏人格概率 +30%
C) 无聊，随便点的                 → 藏人格概率 +50%
```

## 五、匹配逻辑

### Step 1：计算用户 15 维向量

每题根据选项，累计每个维度的加分/减分。20 题全部答完后，得到用户 15 维原始分。

### Step 2：归一化

每个维度范围控制在 **-2 ~ +2** 之间。如果累计超出范围则截断。

### Step 3：向量匹配

计算用户向量与 16 种人格类型的 15 维模板向量之间的**欧几里得距离**（或点积相似度），取距离最小的类型作为主结果。

同时记录**第二接近的类型**作为"你的对立面也有可能是..."的补充展示。

### Step 4：隐藏人格判定

如果触发隐藏题且最终判定概率 > 70%，则覆盖为「空窗期误入者」。

### Step 5：生成结果页

结果页包含：
1. **人格类型名 + 标签**（大号展示，方便截图）
2. **15 维雷达图**（让人感觉"好像真的挺准"）
3. **一段 AI 生成的恋爱画像小作文**（基于维度描述用户类型）
4. **你可能也接近**（第二人格）
5. **适合你的恋爱类型**（哪些类型和你互补）
6. **分享按钮**（一键生成分享图）
