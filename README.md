# 🌿 树洞情绪站

> 一个基于 AI 的性格、焦虑评级与亲密关系评测工具。
> 匿名、私密、免费使用。

## 📋 项目概述

树洞情绪站是一个单页网页应用，帮助用户了解自己的**性格特质**、**焦虑水平**和**亲密关系模式**。用户完成简短测评后，AI 会生成一份有温度、有深度的个人心理报告。

### 三大测评模块

| 模块 | 测量内容 | 理论依据 |
|:----|:--------|:--------|
| 🧩 **性格风格** | 大五人格简化版（外向性/宜人性/尽责性/神经质/开放性） | 大五人格理论 (McCrae & Costa) |
| 😰 **焦虑评级** | 焦虑水平评估 + 典型焦虑模式识别 | GAD-7 改良版，CBT 认知行为模型 |
| 💕 **亲密关系** | 关系健康度 + 互动模式分析 | 戈特曼 5:1 比例理论、依恋理论 (Bowlby) |

## ✨ 功能特色

- **📝 在线测评**：三个独立测评，各 10~15 题，5 分钟内完成
- **🤖 AI 解读报告**：基于测评数据，生成个性化分析 + 实操建议
- **📊 结果可视化**：情绪趋势折线图、性格雷达图、关系健康仪表盘
- **🔒 完全匿名**：不收集手机号、不注册、不追踪
- **💬 情绪树洞**：匿名倾诉 + AI 有温度的回应的增值服务
- **📅 每日打卡**：情绪追踪，帮你发现情绪规律

## 🧠 理论基础

本产品基于以下心理学研究成果，确保测评的专业性和科学性。

| 理论 | 创始人 | 在本产品中的应用 |
|:----|:------|:--------------|
| 依恋理论 | Bowlby, Ainsworth | 亲密关系中的依恋风格分类（安全/焦虑/回避） |
| 戈特曼关系研究 | John Gottman | 5:1 积极互动比例、末日四骑士识别 |
| 社会交换理论 | Thibaut & Kelley | 关系满意度 = 收益 vs 期望，替代选择评估 |
| 自我扩展理论 | Aron & Aron | 关系倦怠检测：亲密关系中的共同成长速度 |
| 大五人格理论 | McCrae & Costa | 五大核心性格维度的简化测量 |
| 认知行为模型 | Beck, Ellis | 焦虑思维模式识别与帮治建议 |

## 🚀 快速开始

### 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JS（单页应用，零依赖）
- **图表**：Chart.js（CDN）
- **AI 报告**：通过 API 调用 AI 模型生成
- **部署**：GitHub Pages / Vercel / Cloudflare Pages（免费）

### 使用方式

1. 打开网页 → 选择你想要测评的模块
2. 回答 10~15 道选择题（约 2~3 分钟）
3. 查看基础测评结果（免费）
4. 如需深度解读 → 扫码支付 → AI 生成专属报告

### 本地开发

```bash
# 克隆项目
git clone https://github.com/your-username/emotion-treehole.git
cd emotion-treehole

# 直接在浏览器打开 index.html 即可
# 无需安装任何依赖
```
