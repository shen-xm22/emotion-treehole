# 🌿 树洞情绪站 — 产品需求文档 (PRD)

> 版本：v1.0 | 日期：2026-05-06 | 状态：初稿

---

## 1. 产品定位

**一句话描述：** 一个匿名、有温度、AI驱动的心理学轻测评与情绪陪伴工具。

**目标用户：**
- **核心用户**：18-35岁，有轻度情绪困扰但对心理咨询有顾虑的年轻人
- **次要用户**：恋爱中想了解彼此关系模式的情侣
- **场景**：深夜焦虑时／失恋后／工作压力大时／想自我探索时

**产品气质：** 温暖、治愈、不评价、不强推、像朋友一样

---

## 2. 核心功能

### 2.1 三大测评模块

| 模块 | 题目数 | 用时 | 测评维度 | 理论依据 |
|:----|:-----:|:---:|:--------|:--------|
| 🧩 性格风格 | 12题 | 2min | 大五人格简化版（外向/宜人/尽责/神经质/开放） | McCrae & Costa |
| 😰 焦虑评级 | 10题 | 1.5min | 焦虑水平分 + 焦虑模式（泛化/社交/躯体化） | GAD-7 + CBT |
| 💕 亲密关系 | 15题 | 3min | 关系健康度 + 依恋风格 + 冲突模式 | 戈特曼 + 鲍尔比 |

### 2.2 AI 解读报告

**免费版**（¥0）：
- 各维度分数与简评（3-5行）
- 雷达图 / 仪表盘可视化

**付费深度版**（¥9.9-19.9）：
- 1500字个性化详细分析
- 情绪问题背后的心理机制解读
- 具体可操作的 3 条改善建议
- 推荐书单（依测评结果动态推荐）

### 2.3 情绪树洞 💬（增值服务）

- 匿名文字倾诉
- AI 以温暖、共情的语气回应（不评判、不荐药）
- 类似"深夜电台"的陪伴感
- ¥9.9 / 次

### 2.4 每日打卡 📅

- 每日 3 个情绪选择题（1 分钟完成）
- 自动生成「情绪周报/月报」趋势
- 连续打卡奖励：每周免费深度解读 1 次

### 2.5 关系匹配（未来规划）

- 两人分别测评 → AI 生成关系合拍报告
- 优势互补分析 + 潜在冲突预警
- 互动改善建议

```
│   └── AI 回应展示

```

## 4. 视觉风格

| 维度 | 要求 |
|:----|:----|
| **整体感觉** | 治愈、温暖、安全、安静——像森林里的小木屋 |
| **主色调** | 暖杏色 #F5E6D3 + 鼠尾草绿 #A8B5A2 + 淡薰衣草 #C9C0D3 |
| **字体** | 圆润、柔和、高可读性（思源柔黑 / Noto Sans SC） |
| **卡片** | 大圆角 20px、柔和阴影、毛玻璃效果可选 |
| **动效** | 微光晕、卡片悬浮、页面过渡柔和、不要花哨弹窗 |
| **图案** | 叶子、星星、云朵、月光等自然元素，手绘风格 |
| **氛围** | 让人想深呼吸的感觉 🌿 |

## 5. 变现模式

| 层级 | 定价 | 受众 |
|:----|:---:|:----|
| 🆓 免费测评+基础结果 | ¥0 | 所有用户（引流） |
| 🌟 深度 AI 解读报告 | ¥9.9 / 次 | 有自我探索需求的用户 |
| 💬 情绪树洞 AI 回应 | ¥9.9 / 次 | 情绪低谷需要陪伴的用户 |
| 📊 月趋势报告 | ¥19.9 / 月 | 长期自我观察的用户 |
| 🏪 企业/学校批量版 | 按需报价 | HR、学校心理健康中心 |
| 💑 关系合拍报告 | ¥29.9 / 对 | 情侣 |

---

## 6. 技术要求

- **架构**：纯前端 SPA（HTML + CSS + JS），无后端
- **API 调用**：通过前端直接调用 AI API，报告生成
- **支付**：微信/支付宝个人收款码（简单方案）
- **数据存储**：LocalStorage（匿名用户） / 离线不存储
- **部署**：GitHub Pages / Vercel（免费）
- **兼容**：Chrome / Safari / 微信内置浏览器
- **性能**：首次加载 < 2s（含 JS 和字体）

---

## 7. 非目标（明确不做）

- ❌ 不做诊断，不替代专业心理咨询
- ❌ 不收集手机号 / 真实姓名 / 身份证
- ❌ 不做社区 / 用户 UGC 功能
- ❌ 不做复杂的后台管理系统
- ❌ 第一期不做双人/情侣同时测评

---

## 8. 里程碑

| 阶段 | 时间 | 交付物 |
|:----|:---:|:------|
| P0 MVP | 第 1 周 | 焦虑评级模块完整上线（题库 + 结果页 + 付费深度报告） |
| P1 扩展 | 第 2 周 | 性格测评 + 亲密关系上线 |
| P2 增值 | 第 3 周 | 情绪树洞 + 每日打卡 + 趋势报告 |
| P3 优化 | 第 4 周 | UI 精调 + 动效打磨 + 文案优化 |

---

> **后记：** 这个产品的核心不是技术，是「温度」。每一个文案、每一个颜色、每一个过渡动效，都应该让用户觉得——我终于有一个安全的地方说话了。
