## 概述

三个测评的设计原则：
1. **有专业依据** — 每个测评都基于经信效度验证的量表
2. **轻量友好** — 题量控制在适中范围，体验轻松不累
3. **树洞风格** — 去掉临床感，保留专业性，题目措辞温暖自然

## 一、🧩 性格风格测评（Personality Snapshot）

### 理论基础
基于 **大五人格模型（Big Five / NEO-FFI）**，提取五个核心维度。原始量表 NEO-FFI 共 60 题，学术简化版 BFI-44 共 44 题，BFI-10 共 10 题。本测评在 BFI-10 的基础上，参考 BFI-44 的经典条目，**每题拆为两个小问题**确保信度，形成 **10 题版本**。

### 五个维度

| 维度 | 低分者（-） | 高分者（+） |
|:----|:----------|:----------|
| **外向性** (E) | 独处充电、内敛深思 | 社交充电、热情外向 |
| **宜人性** (A) | 直接尖锐、独立性强 | 温和友善、乐于合作 |
| **尽责性** (C) | 随性自由、不拘小节 | 计划有序、可靠负责 |
| **情绪稳定性** (N⁻) | 敏感易虑、情绪起伏 | 沉稳淡定、抗压性强 |
| **开放性** (O) | 务实传统、喜欢熟悉 | 好奇创新、拥抱变化 |

### 题目（10题，每题两个陈述，5点李克特量表）

> 指导语：以下是一些描述性格特点的句子，请根据 **你对自己的感觉** 来判断符合程度。每对题目两句话程度可能不同，请分别评分。
> 
> 1 = 非常不符合　2 = 比较不符合　3 = 一般　4 = 比较符合　5 = 非常符合

| 题号 | 维度 | 题目（正向陈述） | 反向计分 |
|:---:|:---:|:----------------|:-------:|
| 1a | E+ | 我在人群中很容易打开话匣子，喜欢和不同的人交流 | |
| 1b | E− | 比起热闹的聚会，我更喜欢一个人安静地待着 | ✅ |
| 2a | A+ | 我通常会先考虑别人的感受，尽量避免冲突 | |
| 2b | A− | 我说话比较直接，不会太在意别人怎么看 | ✅ |
| 3a | C+ | 我做事习惯提前计划，喜欢井井有条 | |
| 3b | C− | 我比较随性，计划赶不上变化是常有的事 | ✅ |
| 4a | N+ | 我很容易多想一件小事，翻来覆去睡不着 | ✅（注：高分=低稳定） |
| 4b | N− | 大多数时候我心态很稳，不容易被小事影响 | |
| 5a | O+ | 我对新事物充满好奇，愿意尝试没做过的事 | |
| 5b | O− | 我比较恋旧，熟悉的东西让我更安心 | ✅ |

> **每题 1a 和 1b 分别计分**，计算方式见下方。
> 
> 每个维度的最后得分 = 正向题得分 + (6 − 反向题得分) ，再取中间值归一化到 1~5。
> 
> 未标注「反向计分」的题目按原始分计算；标注 ✅ 的题目需反向后与同维度正向题平均。

### 计分方式（简化版）

```
外向性 = (1a得分 + (6 − 1b得分)) / 2
宜人性 = (2a得分 + (6 − 2b得分)) / 2
尽责性 = (3a得分 + (6 − 3b得分)) / 2
情绪稳定性 = ((6 − 4a得分) + 4b得分) / 2   ★ 注意这里4a是反向题（高分=焦虑）
开放性 = (5a得分 + (6 − 5b得分)) / 2
```

> 最终每个维度分数范围 1~5 分，保留一位小数。
> 3.0 为中等，>3.5 偏高分，<2.5 偏低分。

### 结果分类与解读

**1. 五维雷达图展示**
展示五个维度的分数，标注"偏高""中等""偏低"区间。

**2. 性格画像（根据得分组合给出标签）**

| 典型组合 | 画像名称 | 一句话描述 |
|:--------|:--------|:----------|
| E↑ A↑ C↑ | 社交靠谱型 | 既有温度又有条理，是朋友心里靠得住的人 |
| E↑ A↓ C↓ | 自由野马型 | 爱交朋友但讨厌被管，活得随性洒脱 |
| E↓ A↑ C↑ | 温柔靠谱型 | 话不多但做事稳，朋友不多但都很铁 |
| E↓ A↓ C↑ | 独行实干型 | 不喜欢社交套路，专注做好自己的事 |
| N↑ O↑ | 敏感探索型 | 内心丰富又敏感，对世界有独特的感知 |
| N↓ O↓ | 稳如泰山型 | 心态稳、喜欢熟悉，是那种让人安心的存在 |
| ... | ... | ... |

> AI 会根据实际五维分数组合，结合用户的具体得分，动态生成一段 200~400 字的性格描述。

### 参考格式（用例）

> **你的性格画像：温柔靠谱型** 🧸
> 
> 你属于话不多但心里有数的那种人。社交对你来说不是必需品，但你在意的人，你会很用心。
> 你做事有计划、有交代，身边的人大概都觉得你「靠谱」。
> 
> *建议*：你有时候对自己要求太高了，偶尔允许自己"摆烂"一下，天不会塌的。
> 
> *情绪关联*：尽责性偏高+情绪稳定性中等——可能是你最大的压力来源，试试把"必须做好"改成"做完了就行"。

## 二、😰 焦虑感自评（Anxiety Check）

### 理论基础
基于 **GAD-7（广泛性焦虑障碍量表）**，Spitzer 等 2006 年编制，中文版经何筱衍等修订。标准 GAD-7 共 7 题，评估过去两周焦虑症状的核心认知-情绪维度。

本测评在保留 GAD-7 全部 7 题的基础上，**补充 5 题**覆盖 GAD-7 未充分涵盖的常见焦虑表现（躯体症状、睡眠影响、社交忧虑、注意力困难、前瞻性担忧），形成更全面的 **12 题版**。同时增加 1 题开放填空（不计分，存入用户档案供后面树洞聊天参考）。

### 题目（共13题，12题评分 + 1题自由回答）

> 指导语：回想一下 **最近两周**，下面这些情况在你身上的出现频率是？
> 
> 0 = 完全没有　1 = 有过几天　2 = 超过一半时间　3 = 几乎每天

| 序号 | 题目 | 来源 |
|:---:|:----|:----|
| **核心 GAD-7（7 题）** | | |
| 1 | 感到紧张、焦虑或心里不踏实 | GAD-7 第1题 |
| 2 | 脑子停不下来，一直担心某些事 | GAD-7 第2题 |
| 3 | 对各种各样的事情担心得太多 | GAD-7 第3题 |
| 4 | 很难真正放松下来 | GAD-7 第4题 |
| 5 | 心里发慌，坐立不安 | GAD-7 第5题 |
| 6 | 容易烦躁或被惹恼 | GAD-7 第6题 |
| 7 | 总觉得有什么不好的事情要发生 | GAD-7 第7题 |
| **补充项（5 题）** | | |
| 8 | 身体出现不舒服——比如心慌、出汗、胃难受、头痛 | 补充-躯体症状 |
| 9 | 晚上躺下后脑子还在转，很难睡着或容易醒 | 补充-睡眠 |
| 10 | 在社交场合总觉得不自在，担心自己说错话做错事 | 补充-社交忧虑 |
| 11 | 很难专心做一件事，注意力总是不由自主地飘走 | 补充-注意力 |
| 12 | 总担心还没发生的事——哪怕那些事根本没影子 | 补充-前瞻担忧 |
| **开放填空** | | |
| **13** | **（开放填空，不计分）最近两周有没有什么具体的事让你特别焦虑？** | 新增 |

### 计分方式

```
总分 = Q1 + Q2 + Q3 + Q4 + Q5 + Q6 + Q7 + Q8 + Q9 + Q10 + Q11 + Q12
范围：0 ~ 36

为了与原 GAD-7 标准分级对齐，同时显示两个分数：
核心分 = Q1~Q7 之和（0~21，保持 GAD-7 可比性）
综合分 = Q1~Q12 之和（0~36，更全面的评估）
```

**结果展示时同时显示两个分数：**
- **核心分**（7题）：与标准 GAD-7 一致，可供纵向对比参考
- **综合分**（12题）：更全面反映焦虑状态，包含躯体/睡眠/社交等维度

### 核心分分级（GAD-7 标准）

| 核心总分 | 等级 | 含义 |
|:-------:|:----|:----|
| 0~4 | 🟢 平稳 | 无明显焦虑困扰，正常范围 |
| 5~9 | 🟡 轻度 | 有一些焦虑感，可能需要关注和调节 |
| 10~14 | 🟠 中度 | 焦虑比较明显，建议主动调整，必要时寻求支持 |
| 15~21 | 🔴 较高 | 焦虑水平较高，建议认真对待 > 触发转介话术 |

### 综合分解读方向

| 综合分 | 树洞解读侧重点 |
|:-----:|:-------------|
| 0~12 | 整体状态不错 |
| 13~20 | 关注个别高分区项，找到具体压力来源 |
| 21~28 | 已在多个方面受到影响，建议系统性调整 |
| 29~36 | 广泛而显著的焦虑影响，触发转介建议 |

### 高危预警（核心分 ≥ 15 或综合分 ≥ 29，或用户主动提到自伤/暴力）

当达到高危标准时，树洞结果页将显示：

> **🌟 树洞的话**：这个结果说明你最近承受了相当大的压力。不管发生了什么事，树洞想告诉你——**你不需要一个人扛**。如果身边有可以信任的人，试着跟他们聊聊。如果需要更专业的帮助，可以拨打心理援助热线（24小时）：
> - **全国心理援助热线：400-161-9995**
> - **北京心理危机干预中心：010-82951332**
> 
> 树洞会一直在，但有些路需要专业的人陪你走。

### AI 解读示例

> **你的焦虑感自评：轻度（核心分6分 / 综合分11分）** 🟡
> 
> 你的核心焦虑分数在正常范围内偏高一些。综合分显示躯体方面（身体不舒服、睡眠）也有一些影响。
> 你提到「最近工作上的事比较多」——可能这才是根源。
> 焦虑本身不是坏事，它在提醒你有些事需要关注。但当它开始影响睡眠或饭量的时候，就是需要按下暂停键的信号。
> 
> *树洞建议*：试试每天留 10 分钟「什么都不做」的时间。你可以发发呆、听一首歌、或者跟树洞说说话。

## 三、💕 亲密关系风格（Relationship Style）

### 理论基础
基于 **ECR（亲密关系经历量表）**，Brennan, Clark & Shaver 1998 年编制，李同归、加藤和生 2006 年中文修订版。原始量表 36 题，分为两个维度各 18 题：

| 维度 | 含义 | 对应原题 |
|:----|:----|:--------|
| **依恋回避** | 对亲密感到不适，倾向于保持情感距离 | 奇数题（1,3,5...35） |
| **依恋焦虑** | 担心被抛弃，过度需要确认和回应 | 偶数题（2,4,6...36） |

### 简化说明

原版 36 题对于用户来说太长。我们提取**每个维度中信度最高的代表性条目各 7 题**，共 14 题。同时将 7 点量表降为 5 点以降低认知负担。

取舍原则：
- 保留因子载荷最高的条目（权威研究中区分度最好的）
- 保留正反两种方向的题目（反向题确保作答质量）
- **严格聚焦浪漫关系**：题目明确指向「恋人/配偶/暧昧对象」，不做泛化

> **重要提示：** 如果用户目前没有恋爱对象，提示语为——「请想象你在一段恋爱关系中，或者回想你过去的一段关系，根据你在那些关系中的真实感受来回答。」

### 题目（14题，5点李克特量表）

> 指导语：当你和 **恋人（或配偶、暧昧对象）** 相处时，你通常在多大程度上同意下面的描述？
> 
> 如果你目前没有恋爱关系，请想象一段关系或回想过去的经历来回答。
> 
> 1 = 完全不同意　2 = 比较不同意　3 = 不确定　4 = 比较同意　5 = 完全同意

| 序号 | 对应原题 | 维度 | 题目 | 反向 |
|:---:|:--------:|:---:|:----|:---:|
| 1 | ECR-01 | 回避 | 我不太喜欢让恋人知道我内心深处的感受 | |
| 2 | ECR-02 | 焦虑 | 我担心会被恋人抛弃 | |
| 3 | ECR-03 | 回避 | 跟恋人亲近对我来说是一件舒服、惬意的事 | ✅ |
| 4 | ECR-06 | 焦虑 | 我担心恋人不像我关心他/她那样关心我 | |
| 5 | ECR-05 | 回避 | 当恋人开始跟我亲近时，我发现自己会不自觉地退缩 | |
| 6 | ECR-10 | 焦虑 | 我常常希望恋人能给我更多回应，但好像总是不够 | |
| 7 | ECR-15 | 回避 | 我能很自然地跟恋人分享我的感受和烦恼 | ✅ |
| 8 | ECR-04 | 焦虑 | 我很担心我们的关系哪天会出问题 | |
| 9 | ECR-33 | 回避 | 我会发现很难完全依赖恋人 | ✅ |
| 10 | ECR-36 | 焦虑 | 当我不在恋人身旁时，我担心他/她会喜欢上别人 | |
| 11 | ECR-13 | 回避 | 当恋人靠得太近时，我会感到紧张 | |
| 12 | ECR-22 | 焦虑 | 我觉得自己一个人也很好，不需要太依赖恋人 | |
| 13 | ECR-07 | 回避 | 当遇到困难时，向恋人寻求帮助对我来说很自然 | ✅ |
| 14 | ECR-34 | 焦虑 | 我经常想要和恋人融为一体，但又怕这样会失去自我 | |

> 反向题：题 **3、7、9、13** 为反向计分（全部在回避维度），计分时需反转。

### 计分方式

```
回避维度 = (Q1 + Q5 + Q11 + (6−Q3) + (6−Q7) + (6−Q9) + (6−Q13)) / 7
焦虑维度 = (Q2 + Q4 + Q6 + Q8 + Q10 + Q12 + Q14) / 7

★ 反向题（全部在回避维度）：Q3、Q7、Q9、Q13，公式 = 6 − 原始分
★ 最终分数范围 1~5 分
```

### 依恋类型判定

将两个维度的分数落入高低区间（以 3.0 为分界）：

```
回避 < 3.0 且 焦虑 < 3.0 → 安全型（Secure）
回避 < 3.0 且 焦虑 ≥ 3.0 → 迷恋型（Preoccupied）
回避 ≥ 3.0 且 焦虑 < 3.0 → 疏离型（Dismissing）
回避 ≥ 3.0 且 焦虑 ≥ 3.0 → 恐惧型（Fearful）
```

### 四种类型的解读

| 类型 | 一句话概括 | 常见表现 | 树洞的话 |
|:---|:---------|:--------|:--------|
| **安全型** 🛡️ | 能靠近也能独立 | 信任关系、能坦诚沟通、不过度担心被抛弃 | 「你在关系中有一种让人安心的稳定。你能给彼此空间，也能在需要时靠近——这是最健康的模式。」 |
| **迷恋型** 💭 | 渴望亲密又怕不够 | 非常在乎关系、容易患得患失、需要高频确认 | 「你给感情的分量很重，这是你的温柔，也是你的软肋。试着相信：对方选择你，不是因为你够好，而是因为你本身就是值得的。」 |
| **疏离型** 🧊 | 我想要自由，别靠太近 | 重视独立、不喜欢依赖、遇到亲密会拉开距离 | 「你习惯了自己扛所有事。独立没有错，但偶尔让别人靠近你一点点，可能没有你想的那么危险。」 |
| **恐惧型** 🌪️ | 既想靠近又怕受伤 | 渴望亲密但害怕被拒绝、常常矛盾、忽远忽近 | 「你心里好像有两个声音：一个说要靠近，一个说会受伤。这种拉扯一定很辛苦。没关系，慢慢来，信任不是一天建成的。」 |

### AI 深度解读（示例）

> **你的亲密关系风格：迷恋型** 💭
> 
> 你的焦虑维度（3.8）明显高于回避维度（2.4）。这意味着你非常在意关系，渴望亲密，但也容易被不安感困扰——可能是对方一条消息回慢了，你就会开始想「是不是我做错了什么」。
> 
> 这不是缺点，而是你的依恋系统在保护你。只是有时候它太敏感了。
> 
> *试试这个*：下次再感到不安的时候，先给自己 20 分钟，做点别的事，再决定要不要跟对方说。很多时候20分钟后那种紧迫感就过去了。

## 测评体验设计要点

### 视觉风格
- 每页只显示 1 题，选项在下方大按钮排列
- 底部轻微进度条，不打扰
- 每题答完后淡入过渡到下一题
- 每题配一个极简的治愈元素（一个小叶子、一个星星、一个小云朵）

### 结果页设计
- 先展示核心结果（分数/类型/雷达图）
- 再展示 AI 解读（打字机效果逐行出现）
- 最后「想和树洞聊聊结果吗？」→ 引导进入聊天

### 免责声明
每个测评结果页底部固定显示：

> ⚠️ 本测评结果仅供参考与自我了解，不构成任何医学诊断。如有持续情绪困扰，建议寻求专业心理咨询。
